# Actividad UD4 - Análisis de Arrays en Java

## Descripción

Este proyecto consiste en un programa escrito en Java que analiza un array de números enteros.

El objetivo de la actividad es trabajar la documentación y la refactorización de código, aplicando las buenas prácticas vistas en la unidad UD4 de la asignatura Entornos de Desarrollo.

---

## Funcionalidad del programa

El programa realiza las siguientes operaciones:

- Busca el número mayor dentro de un array.
- Comprueba si el número mayor se repite.
- Calcula la media de los valores del array.
- Muestra por pantalla si la media es mayor o igual que 5 (aprobado) o no.
